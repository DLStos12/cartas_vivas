Cartas Vivas é um projeto web com foco no fortalecimento da fé cristã, reunindo conteúdos como pregações em vídeo, devocionais, louvores, versículo da semana, acesso à Bíblia online e espaço para pedidos de oração.

## Visão geral

O objetivo do projeto é oferecer uma experiência simples, acolhedora e responsiva para usuários que desejam consumir conteúdo cristão em uma plataforma digital.

## Funcionalidades

- Exibição de pregações em vídeo
- Seção de devocionais
- Seção de louvores
- Player principal incorporado com vídeos do YouTube
- Modal com conteúdo recomendado
- Versículo da semana com reflexão
- Link para Bíblia online
- Formulário para pedidos de oração
- Layout responsivo para desktop, tablet e celular

## Tecnologias utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- YouTube Embed
